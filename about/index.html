本站的一些事

1. 背景

在很久之前的某一天，浏览着网页，感叹着这网站真好看，脑袋突然蹦出个想法，搭建一个属于自己的网站会怎么样呢，那应该会很...cool吧。然而搜索了下，发现搭建网站真复杂，还费钱，当时在还在十二年教育里面，口袋空空如也。随后这个想法和当时很多很多想法一样，无限搁浅...

2. 博客的搭建

最近较为空闲，正好在学习github，决定用github搭建一个属于我自己，不对外开放的个人博客。

2.1 使用技术

比较懒，且确实不太会这个，非科班出生，所以选择Fork别人的框架和内部装饰，后续有时间，再进行改动。

hexo-theme-butterfly + 一些魔改。

在技术的选择上，使用Hexo搭建一个静态博客。

Hexo是一个简单、高效且强大的博客框架，只需要一些环境，就可以在短时间内搭建出一个简易的博客。随后再添加hexo-theme-butterfly主题

当然，如果你也想搭建一个博客，但是不喜欢这个主题，可以去GitHub上搜索 hexo theme 即可，可以找到许多好看的主题。

2.2 参考链接

如果你也想搭建一个博客，需要准备以下环境：Node.js、Git，其中Git需要会使用。如果你还想进行魔改，那么一定的HTML + CSS + JS的知识是不可缺少的，至少能看得懂。

又或者你没有时间/等不及想要先搭建个网站先看看，只需要了解Github的基本应用，Fork别人的模板就可以了。
